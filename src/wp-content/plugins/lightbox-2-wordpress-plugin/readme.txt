Plugin used to overlay images on the current page into neat Javascript-powered overlay popups. From settings you can choose background-color, overlay opacity, resize speed and automatically add the lightbox effect to your post's images.

== Description ==

This plugin includes the new Lightbox JS v2.04 javascript written by Lokesh Dhakar and got transformed into a Wordpress Plugin by me.
From settings you can choose background-color, overlay opacity, resize speed and automatically add the lightbox effect to your post's images.
Use the title attribute if you want to show a caption.
[Plugin Homepage](http://www.4mj.it/lightbox-js-v20-wordpress/ "Plugin Homepage")

== Installation ==

**Upgrade**

1. De-activate the plugin.
1. Delete the old version `lightbox-2-wordpress-plugin` directory

**Install**

1. Unzip the `lightbox-2-wordpress-plugin.zip` file.
1. Upload "lightbox-2-wordpress-plugin" folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress, go to Setting->Lightbox
3. Add a rel="lightbox" attribute to any link tag to activate the lightbox or rel="lightbox[roadtrip]" for image set
